# Features & Tool Suite | DevBox

DevBox is a comprehensive suite of developer tools categorized by function. Every tool operates 100% in your browser for absolute privacy and maximum speed.

## 🚀 Core Features
- **100% Client-Side**: No data ever leaves your browser.
- **Glassmorphism Design**: Ultra-premium aesthetic with Dark/Light mode support.
- **Persistent State**: Your settings and recent data are saved locally.
- **Offline Capable**: PWA support for use without internet.
- **High Performance**: Built with React 19 and Vite 7.

## 📚 Complete Tool List

### 🛠️ Development & Engineering
- **API Tester**: Debug REST & HTTP requests with ease.
- **Architecture Studio**: Immersive system design with Mermaid syntax and Blueprint grid.
- **Regex Tester**: Real-time pattern matching with capture groups.
- **JWT Tool**: Decode, verify, and generate JSON Web Tokens locally.
- **CSS Flexbox & Grid**: Visual layout builders with instant code export.
- **Bezier Curves**: Interactive cubic-bezier easing function designer.
- **URL Tool (Pro)**: Structural parser and segment encoder/decoder.
- **User Agent Parser**: breakdown UA strings into device/browser hints.
- **HTTP Status Codes**: Instant reference for status codes.
- **Cron Viewer**: Human-readable explanations for cron expressions.
- **Base Converter**: BigInt support for bases 2-36.
- **GitBox**: GitHub README generator and Profile Stats viewer.

### 📊 Data & Encoding
- **JSON Formatter**: Prettify, validate, and minify JSON.
- **JSON String Escape**: Handle JSON literals and escape sequences.
- **SQL Formatter**: Professional SQL query beautification.
- **CSV to JSON**: Instant data conversion.
- **XML & YAML Tools**: Format, validate, and bridge formats.
- **Base64 Master**: Encode/Decode text and files instantly.
- **Hex Encoder**: Convert UTF-8 text to hex bytes.
- **Unicode Escape**: Handle specialized character encoding.
- **HTML Entity**: Cross-convert between characters and entities.

### 🔐 Security & Utilities
- **Password Toolkit (Pro)**: Entropy analysis, breach checking, and secure generation.
- **Hash Calculator**: Compute SHA-1, SHA-256, SHA-512 locally.
- **HMAC Signer**: Generate secure signatures using WebCrypto.
- **AES-256 Crypto**: Secure encryption/decryption with PBKDF2.
- **File Integrity**: Compute checksums for local files.
- **Token Generator**: bulk UUID v4/v7 and NanoID generation.
- **Subnet Calculator**: IPv4 CIDR analysis and usable range calculation.
- **IP & DNS Suite**: Lookup IP (RDAP), DNS (DoH), MAC vendor, and validate addresses.

### ⏱️ Time & Date
- **Timestamp Tool**: High precision Unix/Epoch converter (ms, μs, ns).
- **ISO 8601 Parser**: Standardized date string breakdown.
- **Timezone Converter**: Synchronize moments across global zones.
- **World Clock**: Real-time tracking of multiple timezones.
- **Duration Converter**: bridge units from ms to days.
- **Age Calculator**: Exact age with days-until-birthday tracking.

### 🎨 Media & Design
- **Image Master (Pro)**: Compress, resize, and crop images with a pro lens.
- **Format Converter**: switch between PNG, JPG, and WebP locally.
- **PDF Toolkit**: Merge, Split, PDF-to-Image, and Image-to-PDF.
- **Color Toolkit (Pro)**: WCAG contrast audit, palette synthesis, and format conversion.
- **QR Generator**: High-quality QR codes with logo support.
- **Gradient Designer**: Linear and radial CSS background architect.

### ✍️ Text & Advanced Utilities

- **Text Diff**: visual comparison of two text blocks.
- **Case Converter**: swap between camel, snake, kebab, and pascal cases.
